# MobileGuard

## Project Overview

MobileGuard is a premium mobile accessories online store, offering a modern and responsive storefront for users to explore and purchase a wide range of high-quality mobile accessories. The website features a clean design, a smooth shopping experience, and an engaging user interface to facilitate easy navigation through different product categories and collections.

## Installation

To get started with MobileGuard, you can follow these simple steps:

1. **Clone the repository:**
   ```
   git clone https://github.com/your-username/mobileguard.git
   ```

2. **Navigate to the project directory:**
   ```
   cd mobileguard
   ```

3. **Open the demo HTML file in your browser:**
   You can simply double-click on `demo.html` or open it using a web browser (e.g., Chrome, Firefox).

## Usage

Once the page is loaded in your browser, you can explore the following features:

- **Home:** An introduction to the store and premium products.
- **Shop:** A selection of featured products available for purchase.
- **Collections:** Categories of products including Tempered Glass, Phone Covers, and Accessories.
- **Newsletter:** A subscription form to stay updated on new products and offers.
- **Footer:** Quick links and contact information.

You can add products to your cart and interact with the UI, which includes a smooth scroll navigation and an alert for subscription confirmation or adding products to the cart.

## Features

- Responsive design compatible with mobile and desktop devices.
- Clear navigation menu for easy access to different pages.
- Dynamic categories and featured products section.
- Newsletter subscription form.
- Interaction alerts for adding products to the cart and quick view.
- Includes CSS styling for modern aesthetics.

## Dependencies

The project currently does not use any JavaScript frameworks or libraries, ensuring that it runs with plain HTML, CSS, and JavaScript without any additional dependencies. 

However, it relies on Google's **Inter** font for styling, which is loaded via a link in the HTML file.

## Project Structure

```
mobileguard/

```

The project consists of a single HTML file (`demo.html`), which contains:
- HTML structure for the website, including sections for header, hero banner, product categories, featured products, a newsletter section, and footer.
- CSS styles for layout and responsiveness directly embedded in the HTML file.
- JavaScript for interactivity like smooth scrolling and button functionalities.
